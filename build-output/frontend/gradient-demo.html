🔧 修复说明

  • 问题原因：TailwindCSS 的渐变类在某些情况下可能不被正确生成或应用
  • 解决方案：使用内联 CSS 样式 style={{background: 'linear-gradient(...)'}
  • 优势：确保渐变效果在所有浏览器和环境下都能正常显示
  • 交互性：添加了鼠标悬停事件来动态改变渐变颜色
